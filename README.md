# InitialSpring

Proyecto de ejemplo con **Spring Boot** que implementa un controlador REST básico, un servicio y un repositorio para la gestión de usuarios.

---

## 🛠 Tecnologías utilizadas

- Java 17
- Spring Boot 3
- Maven
- H2 Database (en memoria)
- Spring Data JPA
- Git y GitHub
